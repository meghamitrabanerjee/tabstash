Secure private vault

Every request is verified with a JWT issued to your account at login. Passwords are bcrypt-hashed server-side. Your stash is completely private — nobody else can access your saved links.

Visual grid dashboard

Scan your entire reading list at a glance. Each saved link becomes a card with a real thumbnail and title — no walls of blue hyperlinks.

Full-text search

Find any saved link in under a keystroke. Search spans titles and URLs across your entire stash — no more scrolling through hundreds of bookmarks.

Smart fallback images

If a page has no og:image, TabStash auto-generates a clean branded placeholder — your grid always looks polished, never broken or empty.

FAQ

Got questions?

Everything you need to know about TabStash. Can't find your answer? Email us.

What's the tech stack?
The backend is FastAPI and Python, using SQLite for storage, Passlib/Bcrypt for password security, and BeautifulSoup4 for scraping. The frontend is vanilla HTML + JavaScript. Fully functional end-to-end — not a mockup.
Can anyone else see my saved links?
No. Every API request is validated against a JWT (JSON Web Token) issued to your account at login. The backend rejects any request without a valid token — your stash is completely private.
What if a page has no preview image?
Our scraper looks for standard OpenGraph (og:image) meta tags. If none is found, TabStash renders a clean branded placeholder so your grid stays visually consistent.
Is this a real, working app?
Yes — TabStash is a fully functional portfolio project demonstrating real backend logic, web scraping, and secure authentication. All features work end-to-end. It's not a mockup or prototype.
Is it actually free?
Yes, completely free. No credit card required, no hidden tiers, no usage limits during the public beta. Create an account and start stashing immediately.
